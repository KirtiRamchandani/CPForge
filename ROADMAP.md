# CP Forge Roadmap

CP Forge grows by shipping useful local-first workflows before optional integrations.

## Phase 1 - Repo Foundation

- Monorepo setup
- README, ROADMAP, PRIVACY, CONTRIBUTING, SECURITY, CODE OF CONDUCT
- Core schemas
- Static roadmaps
- Static sheets
- Export templates

## Phase 2 - CLI MVP

- `init`
- `launch`
- `roadmap`
- `sheet`
- `doctor`
- `next`
- `review`
- `upsolve`
- `mistakes`
- `mindmap`
- `chart`
- `export`
- `portfolio`

## Phase 3 - API And Data Adapters

- Codeforces public API with cache, backoff, and one-request-per-two-second queueing
- Codeforces DOM parser
- LeetCode DOM parser
- Optional isolated LeetCode GraphQL adapter
- Custom CSV/JSON imports

## Phase 4 - Dashboard MVP

- Home
- Today
- Roadmaps
- Mindmap
- Sheets
- Charts
- Mistakes
- Reviews
- Upsolve
- Settings

## Phase 5 - Chrome Extension MVP

- Codeforces sidebar
- LeetCode sidebar
- Notes
- Status tracking
- Mistake reminders
- Timer
- Mindmap popup
- Import/export

## Phase 6 - Recommendation Engine

- Weak topic detection
- Next problem recommendation
- Review recommendations
- Upsolve priority
- Company plan adjustment
- Explanations for every recommendation

## Phase 7 - VS Code Extension

- Start problem
- Run samples
- Track timer
- Add notes
- Add mistakes
- Mark solved
- Show checklist

## Phase 8 - Full Polish

- Demo GIFs and terminal demo script (`scripts/terminal-demo.sh`)
- Screenshots and SVG previews
- Release workflow (npm + extension artifacts)
- Role-based and time-based roadmaps
- Company readiness radar in dashboard
- Sheet row actions in dashboard
- Chrome popup skill tree
- AI-optional mode documented (local-first default)
- Hall of fame and ecosystem docs

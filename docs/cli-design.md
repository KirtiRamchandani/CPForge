# CLI Design

The CLI is the main entry point.

```bash
cp-forge launch --cf handle --leetcode handle --goal amazon --days 45
```

CLI commands are rule-based and deterministic. The CLI should always explain why a recommendation was made.

Core commands:

- `init`
- `launch`
- `sync`
- `roadmap`
- `sheet`
- `doctor`
- `next`
- `review`
- `upsolve`
- `mistakes`
- `contest`
- `mindmap`
- `chart`
- `export`
- `import`
- `portfolio`
- `dashboard`
- `reset`

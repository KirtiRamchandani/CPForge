# Dashboard Design

The dashboard is the visual command center for CP Forge.

Pages:

- Home
- Today
- Roadmaps
- Mindmap
- Sheets
- Progress
- Charts
- Weaknesses
- Mistakes
- Upsolve
- Reviews
- Contests
- Companies
- Platforms
- Notes
- Portfolio
- Settings

The dashboard should feel fast, dark-mode first, keyboard friendly, and dense enough for serious daily use.

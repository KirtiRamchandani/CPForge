# Chrome Extension Design

Name: CP Forge Companion

The extension injects a local-first sidebar on supported problem pages. It stores settings in `chrome.storage.local` and problem data in IndexedDB.

MVP platforms:

- Codeforces
- LeetCode

Core features:

- Problem sidebar
- Status tracking
- Notes
- Mistakes
- Timer
- Review date
- Upsolve status
- Submit checklist
- Link badges
- Popup daily plan
- Options import/export

# VS Code Extension Design

Name: CP Forge VS Code

Commands:

- CP Forge: Start Problem
- CP Forge: Run Samples
- CP Forge: Add Mistake
- CP Forge: Add Note
- CP Forge: Mark Solved
- CP Forge: Mark Attempted
- CP Forge: Send to Review
- CP Forge: Open Notes
- CP Forge: Show Checklist
- CP Forge: Show Weak Topic Warning
- CP Forge: Export Session

The extension syncs with the local `.cpforge/` workspace.

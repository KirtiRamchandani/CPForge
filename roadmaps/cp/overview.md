# Competitive Programming Roadmap

## Newbie: 800-1000

Expected skill: clean implementation, math basics, simple greedy.

## Pupil: 1000-1200

Expected skill: prefix sums, sorting, constructive observations, basic strings.

## Specialist: 1200-1500

Expected skill: graph basics, binary search, DP basics, contest upsolve discipline.

## Expert: 1500-1800

Expected skill: stronger proof, shortest paths, data structures, advanced DP.

## Candidate Master: 1800-2100

Expected skill: topic mixing, fast debugging, strategy, editorials as review.

## ICPC Advanced

Expected skill: team strategy, role ownership, notebook fluency, penalty management.

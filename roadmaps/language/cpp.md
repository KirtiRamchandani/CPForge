# C++ Roadmap For DSA And CP

- Use `ios::sync_with_stdio(false); cin.tie(nullptr);`
- Prefer `long long` for arithmetic that can overflow `int`.
- Know `vector`, `array`, `string`, `map`, `unordered_map`, `set`, `priority_queue`, `deque`.
- Watch comparator strict weak ordering.
- Avoid recursion depth crashes on deep trees.

# Competitive Programmer Roadmap

For rating growth on Codeforces (800 → 1600+).

## Rating bands
| Band | Focus | Weekly volume |
|------|-------|---------------|
| 800–1100 | Implementation, brute force, greedy basics | 25 problems |
| 1100–1400 | Binary search, two pointers, DSU intro | 20 + 2 contests |
| 1400–1600 | DP, graphs, math, constructive | 15 + 3 contests |
| 1600+ | Advanced DP, flows, FFT (optional) | Contest-first |

## Weekly structure
- Mon–Thu: topic blocks from `cp-forge sheet --cp-level specialist`
- Fri: upsolve day — `cp-forge upsolve priority`
- Sat–Sun: live or virtual contests — `cp-forge contest --upcoming`

## Sync & track
```bash
cp-forge sync --cf your_handle --all
cp-forge mindmap --goal cp --export html
cp-forge chart --type weakness --export png
```

## Upsolve discipline
Every contest wrong/unsolved problem → upsolve queue within 48 hours.

## Mindset
- Log TLE/WA patterns in mistake bank
- Review solved problems on days 1, 3, 7, 14, 30
- Use Chrome extension contest cockpit during live rounds

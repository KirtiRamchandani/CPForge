# ICPC Contestant Roadmap

For team training, regional prep, and World Finals ambition.

## Team roles
- **Coder A**: graphs, flows, geometry
- **Coder B**: DP, strings, data structures
- **Coder C**: math, greedy, implementation speed

## Season plan (16 weeks)
1. **Weeks 1–4**: Classic ICPC sheet — `cp-forge pack add icpc-classics`
2. **Weeks 5–8**: Regional past problems + virtual team contests
3. **Weeks 9–12**: Weak topic rotation per member
4. **Weeks 13–16**: Full simulation (5h blocks, print statements, penalty tracking)

## Team workspace
```bash
cp-forge team init --name "University ICPC"
cp-forge team add-problem --url <contest-problem>
cp-forge export --format sheets
```

## Contest day
- Chrome extension contest cockpit on Codeforces/AtCoder
- Shared upsolve list synced via `cp-forge extension-import`
- Post-contest: `cp-forge mistakes add` for each wrong submission

## Problem sources
- Codeforces gym, AtCoder ABC/ARC, CSES advanced, Kattis

## Acceptance
- Solve 200+ ICPC-tagged problems as a team
- 10+ full 5-hour simulations before regionals

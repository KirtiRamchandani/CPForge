# Student Roadmap

For undergraduates balancing coursework, internships, and first DSA foundations.

## Week 1–2: Foundations
- Arrays, strings, hashing, two pointers
- 15 easy problems with notes in `.cpforge/notes/`
- Run `cp-forge doctor` weekly

## Week 3–4: Core structures
- Stacks, queues, linked lists, binary search
- 20 medium problems; log mistakes after every WA

## Week 5–8: Interview core
- Trees, graphs, sliding window, basic DP
- One virtual contest per week: `cp-forge contest --rating 1000`

## Week 9–12: Placement prep
- Company sheet: `cp-forge sheet --company amazon --limit 50`
- Spaced reviews: `cp-forge review today`
- Export portfolio: `cp-forge portfolio --github-readme`

## Daily rhythm (90 min)
1. Warmup (15 min) — easy review
2. Main block (45 min) — one weak topic
3. Review (15 min) — spaced repetition
4. Reflection (15 min) — one mistake + edge case

## CLI
```bash
cp-forge launch --goal dsa --days 90 --offline
cp-forge roadmap --goal dsa --days 90
cp-forge sheet --topic arrays --limit 30
```

# 45-Day Standard Plan

Balanced path for most placement and internship candidates.

## Milestones
- **Day 7**: 40 problems logged, mistake bank active
- **Day 14**: All core patterns touched once
- **Day 21**: First full mock + upsolve backlog < 5
- **Day 30**: Company sheet 60% complete
- **Day 45**: Portfolio export + readiness > 70%

## Weekly rotation
1. Arrays / hashing / two pointers
2. Trees / graphs
3. DP / greedy
4. Binary search / math / heap
5. Mixed review + contest
6. Company-specific + mocks

## Commands
```bash
cp-forge launch --goal amazon --days 45
cp-forge weekly  # via dashboard Progress tab
cp-forge sheet --company microsoft --limit 30
cp-forge portfolio --html
```

## Review cadence
After every solve: schedule reviews at 1, 3, 7, 14, 30 days.

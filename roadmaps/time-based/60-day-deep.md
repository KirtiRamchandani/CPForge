# 60-Day Deep Prep

For candidates who want mastery, not just pattern recognition.

## Structure
- **Days 1–20**: Foundation + breadth (all major topics)
- **Days 21–40**: Depth on 3 weak areas from `cp-forge doctor`
- **Days 41–50**: Multi-company sheets + system design notes
- **Days 51–60**: Contests, upsolve, mock interviews

## Volume targets
- 120+ unique problems solved
- 30+ upsolved
- < 10 recurring mistake categories

## Advanced blocks
- Hard DP (knapsack variants, digit DP intro)
- Graph algorithms (Dijkstra, topo sort, SCC intro)
- Segment tree / Fenwick (CP track)

```bash
cp-forge launch --goal placement --days 60
cp-forge mindmap --export html
cp-forge chart --type progress --export svg
```
